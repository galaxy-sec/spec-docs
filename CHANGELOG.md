# Changelog

## 0.7.2

* localize 指定数值文件
* 数值文件支持环境变量

```
Usage: ds-mod localize [OPTIONS]

Options:
  -d, --debug <DEBUG>  [default: 0]
      --log <LOG>      config log ; eg: --log  cmd=debug,parse=info
      --value <VALUE>  use vlaue file; eg: --value cicd_value.yml
  -h, --help           Print help
```
